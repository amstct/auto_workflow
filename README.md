## 项目简介

**auto_workflow** 是一个基于 Coze 平台的自动化工作流生成工具。该项目旨在根据用户的描述，自动生成工作流，并提供建议和 JSON 表达。

## 项目功能

- **自动生成工作流：** 根据用户的描述，自动生成工作流的建议。
- **调用多种节点：** 支持调用插件、LLM（大语言模型）、代码等多种节点。
- **JSON 表达：** 生成工作流的 JSON 表达，方便集成和部署。
